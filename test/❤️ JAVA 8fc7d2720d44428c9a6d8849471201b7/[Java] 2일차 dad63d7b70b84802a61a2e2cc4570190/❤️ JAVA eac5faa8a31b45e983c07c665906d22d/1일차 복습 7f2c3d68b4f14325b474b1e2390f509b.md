## ✏️  Java Spec

1. J2SE (Java Standard Edition)
    - JAVA : OOP(상속, 생성자 ...) + Advanced Feature(예외처리, Thread ...)
    - JDBC : SQL + Java DB
2. J2EE (Java Enterprise Edition)
    - jdk 설치에 더불어 서버 설치가 필요하다
    - Servlet / JSP
    - Front UI 를 위한 기술(HTML, CSS5, JavaScript) 도 필요하다
    - Spring Framework : DI, MyBatis, Spring MVC
