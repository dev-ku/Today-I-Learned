## AWS에 배포하기
- AWS에 프로그램을 배포하고, 로컬에서 접근해보자

<br>

### 배포할 프로그램 Export
- 배포할 프로젝트를 우클릭하고`WAR file` 로 압축해서 Export한다
	- Destination: 경로/ch2.war → Finish
	- `war`: JAVA application을 하나의 파일로 압축하기위한 확장자 (확장자를 zip으로 변경하면 압축된 파일을 확인할 수 있다)
